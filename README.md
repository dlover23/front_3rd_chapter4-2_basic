# 바닐라 JS 프로젝트 성능 개선

- url: http://hanghae-test-bucket.s3-website-ap-southeast-2.amazonaws.com/

## 성능 개선 보고서

### 초기 문제점

#### 1. 이미지 비효율성

- 구형 이미지 포맷 (JPG)
- 최적화되지 않은 해상도
- 불필요한 즉시 로딩

#### 2. 폰트 로딩 문제

- 웹 폰트 다운로드 지연
- 텍스트 가시성 저하

#### 3. 자바스크립트 실행 과제

- DOM 렌더링 전 스크립트 실행
- 성능 병목 현상

### 최적화 전략

#### 1. 이미지 최적화

- WebP 포맷 변환
- 지연 로딩 구현
- 반응형 이미지 해상도

#### 2. 폰트 최적화

- TrueType (TTF) 폰트 최적화
- 텍스트 가시성 개선

#### 3. 자바스크립트 로딩 개선

- 비동기 로딩 구현
- DOMContentLoaded 이벤트 활용
- 초기 렌더링 속도 개선
